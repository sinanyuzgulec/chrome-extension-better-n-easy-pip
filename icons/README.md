# Icon Placeholder

İkonlar için bu klasörü kullanın. Aşağıdaki boyutlarda PNG formatında ikonlar eklemeniz gerekiyor:

- `icon16.png` - 16x16 piksel
- `icon48.png` - 48x48 piksel  
- `icon128.png` - 128x128 piksel

## Hızlı İkon Oluşturma

Ücretsiz online araçlar kullanabilirsiniz:
- [Canva](https://www.canva.com/) - Kolay tasarım aracı
- [Figma](https://www.figma.com/) - Profesyonel tasarım
- [IconScout](https://iconscout.com/) - Hazır icon kütüphanesi

## Tasarım Önerileri

🎨 **Tema**: PIP (Picture-in-Picture) temasını yansıtan bir tasarım
- Video player ikonu
- Ekran içinde ekran görseli
- Mor/pembe gradient renk şeması (extension temasına uyumlu)

📐 **Format**:
- PNG formatında
- Şeffaf arka plan
- Net ve basit tasarım
- Küçük boyutlarda bile okunabilir olmalı
